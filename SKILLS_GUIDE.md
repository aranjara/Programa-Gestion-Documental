**Examples:**
- "Use @python-performance-optimization to check this script."
- "Use @tailwind-design-system to style the new component."
- "Use @security-auditor to review the database access logic."

All skills currently in `.agent/skills/` have been filtered for **"safe"** or **"none"** risk levels according to the library index. This ensures that no offensive or potentially harmful tools are active.
